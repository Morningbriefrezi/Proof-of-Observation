# STELLAR — Your AI-Powered Window to the Cosmos

The all-in-one astronomy app for telescope owners and space enthusiasts. 7-day sky forecast, real-time planet tracker, AI space companion, and real product marketplace — powered by invisible Solana infrastructure.

Built by [Astroman](https://astroman.ge) — Georgia's first astronomy store · 60K followers · Physical store in Tbilisi

## Features
- **7-Day Sky Forecast** — Hourly cloud cover, seeing, transparency. Plan observations days ahead.
- **Planet & Object Tracker** — Real-time positions, rise/transit/set times, visibility windows.
- **AI Space Companion** — Personal astronomer powered by Claude API. Speaks English and Georgian.
- **Astroman Marketplace** — Real telescopes, moon lamps, star maps. Pay with card or SOL.
- **Discovery Attestations** — Verified observations minted as compressed NFTs on Solana.
- **Invisible Crypto** — Privy embedded wallets. Email signup. No seed phrases. Gasless.

## Stack
- Next.js 15 + React 19 + TypeScript
- Tailwind CSS 4
- Privy SDK (embedded Solana wallets, auth)
- Claude API (AI companion, EN + Georgian)
- Open-Meteo + astronomy-engine (sky data)
- Metaplex Bubblegum (compressed NFTs)
- Solana Web3.js + Solana Pay
- Supabase (backend)
- next-intl (English + Georgian)

## Run locally
```bash
npm install
cp .env.example .env.local
npm run dev
```
